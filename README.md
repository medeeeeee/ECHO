# ECHO (Environmental Challenges and Habits Online)
A web app that helps you build good habits to take care of the environment. Your main mission is to take care of a small plant by completing tasks that help the planet, you can complete part of  them online and in real life. 

## Requirements
-Python 3.12
-flask (Flask, render_template, request, redirect, session)
-flask_sqlalchemy (SQLAlchemy)
-jinja2
-datetime (date, datetime, timedelta)
-Random
